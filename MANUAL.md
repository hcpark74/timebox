# Schedule App 사용자 매뉴얼 (User Manual)

Schedule App(스케줄 앱)의 사용 방법을 안내합니다. 이 앱은 **일정 관리, 태스크 우선순위 설정, 그리고 하루의 타임라인 관리**를 직관적으로 할 수 있도록 설계되었습니다.

---

## 1. 🏠 주요 화면 구성

앱은 크게 4가지 페이지로 구성되어 있습니다.

### 📅 Calendar (홈 화면)
- **주간 달력**: 현재 주차의 날짜를 보여줍니다. 오늘 날짜는 색상으로 표시됩니다.
- **날짜 선택**: 상단의 요일을 클릭하면 해당 날짜의 일정을 필터링해서 볼 수 있습니다.
- **스케줄 목록**: 선택된 날짜에 예정된 일정(Scheduled Task) 목록을 시간순으로 보여줍니다.

### ✅ Schedule Task (할 일 관리)
- **Today's Priorities (오늘의 우선순위)**: 오늘 반드시 처리해야 할 중요 업무 목록입니다.
- **Backlog (대기 중인 업무)**: 나중에 처리하거나 아직 우선순위가 정해지지 않은 업무입니다.
- **태스크 추가**: 하단의 입력창에 할 일을 적고 `Add` 버튼을 누르면 새로운 태스크가 생성됩니다.
- **삭제**: 태스크 오른쪽의 🗑️ 아이콘을 누르면 삭제할 수 있습니다.
- **순서 변경 (Drag & Drop)**: 태스크를 길게 눌러 위아래로 드래그하면 순서를 바꾸거나, Priorities와 Backlog 사이를 이동할 수 있습니다.

### 📝 Schedule Your Task (스케줄링)
- `Let's Schedule` 버튼을 누르면 진입합니다.
- **배정되지 않은 태스크**: 왼쪽 목록에 아직 시간이 정해지지 않은 태스크들이 표시됩니다.
- **타임 슬롯**: 오른쪽에는 시간대(08:00 ~ 20:00)가 표시됩니다.
- **스케줄링 방법**: 왼쪽의 태스크를 드래그하여 오른쪽의 원하는 시간대에 놓으세요(Drop). 자동으로 해당 시간에 일정이 배정됩니다.

### ⏱️ Timeline (타임라인 뷰)
- 하루의 흐름을 시각적으로 확인할 수 있는 타임라인입니다.
- 스케줄링된 태스크가 시간대별 블록으로 표시됩니다.
- **완료 체크**: 태스크 블록의 체크박스(☐)를 클릭하면 완료 상태(Done)로 변경되며, 색상이 칠해집니다.

### 👤 Profile (프로필 및 통계)
- **사용자 정보**: 프로필 이미지와 인사말이 표시됩니다.
- **카테고리별 통계**: Personal, Work 등 카테고리별로 등록된 태스크의 개수를 확인할 수 있습니다.

## 2. 🚀 주요 기능 가이드

### 1️⃣ 새로운 할 일 추가하기
1. 하단 메뉴의 **+ (FAB 버튼)**을 클릭하거나 `Schedule task` 페이지로 이동합니다.
2. 입력창에 할 일 제목(예: "마트 장보기")을 입력합니다.
3. **Add** 버튼을 클릭하면 Backlog 또는 Priorities에 추가됩니다.

### 2️⃣ 일정을 시간대에 배정하기 (스케줄링)
1. `Schedule task` 페이지에서 하단의 **Let's Schedule** 버튼을 클릭합니다.
2. 왼쪽 목록에서 원하는 태스크를 선택합니다.
3. 오른쪽 타임라인의 **원하는 시간(예: 05:00 PM)**으로 드래그해서 놓습니다.
4. 하단의 **Schedule** 버튼을 눌러 `Timeline` 페이지에서 결과를 확인합니다.

### 3️⃣ 업무 완료 처리하기
1. `Timeline` 페이지 또는 `Calendar` 페이지의 스케줄 목록에서 해당 태스크를 찾습니다.
2. 태스크 옆의 **체크박스**나 **상태 아이콘**을 클릭합니다.
3. 상태가 **Pending** → **Done(Completed)** 으로 변경됩니다.

## 3. 📱 앱 설치 방법 (PWA)

이 앱은 **PWA(Progressive Web App)** 기술이 적용되어 있어, 앱스토어 없이도 스마트폰에 설치할 수 있습니다.

### 아이폰 (iOS Safari)
1. Safari 브라우저로 접속합니다.
2. 하단의 **공유 버튼** (네모에서 화살표가 위로 나가는 아이콘)을 누릅니다.
3. 메뉴에서 **'홈 화면에 추가'**를 선택합니다.
4. 오른쪽 상단의 **'추가'**를 누르면 홈 화면에 아이콘이 생깁니다.

### 안드로이드 (Chrome)
1. Chrome 브라우저로 접속합니다.
2. 상단 오른쪽의 **메뉴 버튼 (점 3개)**을 누릅니다.
3. **'앱 설치'** 또는 **'홈 화면에 추가'**를 선택합니다.
4. 안내에 따라 설치하면 앱 서랍 또는 홈 화면에 아이콘이 생깁니다.

---

## 4. 💾 데이터 저장

- 이 앱은 **로컬 스토리지(Local Storage)**를 사용하여 데이터를 사용자의 기기에 직접 저장합니다.
- 인터넷 연결이 없어도 기본적인 기능을 사용할 수 있습니다.
- **주의**: 브라우저 캐시를 삭제하거나 시크릿 모드에서 사용할 경우 데이터가 사라질 수 있습니다.
